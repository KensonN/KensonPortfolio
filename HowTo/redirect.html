The Redirect Page

Since I had two different topics for my hardware page, I felt it was necessary to split those two topics into different pages. However, since it would be a little confusing for a user browsing the website, I decided to dedicate a page redirecting to both these pages

The Background

On this page, I have an image set as the background. To do this, I set the class of the body to “hardware”. I used the CSS “background-image” property to set the image to “../sc/VectorPhone.jpg”(stored with the HTML and CSS files). Other than that, the other CSS properties in the “hardware” class will be talked about later.

HTML: 

<body class="hardware">

CSS: 

hardware {
background-image: url(../sc/VectorPhone.jpg)
}

Background Opacity

I decided that I wanted to put an opaque background color over my background image. To do this, I created a div inside the <body> tag, with a class “layer”. The only purpose of the div is to put a background color over the background image. I used the “background-color” property to set the background color to rgba(0,0,0, 0.5). This would create a black but transparent color on the background, making the background image look darker. To position this layer to cover the whole page, I used the “position”, “top”, “left”, “width”, and “height” properties. I set the “position” property to position the background color. I then used the “top”,”left”, “width”, and “height” properties to stretch this background color across the whole page. I set the “top” and “left” properties to 0 so that there would be no space between the top and left sides of the page and the div containing the background. I then set the “width” and “height” class to 100% to stretch it across the whole page.

HTML: 

<body class="hardware">
    <div class="layer">
    </div>
</body>

CSS: 

.layer {
    background-color: rgba(0, 0, 0, 0.7);
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
}

Buttons

On this page, I needed a quick way for the user to easily see how to get to the two hardware pages. To do this, I created two buttons that redirected to one page each. I first created a div with the class “buttons”. In this div, created two anchor tags (<a>) to redirect to each page. One anchor tag had the text “History” while the other had the text “Parts”. Each of these anchor tags have the class “HistoryButton”.However, these anchor tags did not look like buttons. To make them look like buttons, I needed to use CSS.

HTML: 

<body class=hardware>
    <div class="layer">
        <div class=buttons>
            <a href="history.html?v=1.01" class="HistoryButton">
                History
            </a>
            <a href="parts.html" class="HistoryButton">
                Parts
            </a>
        </div>
    </div>
</body>

In my CSS, I first used the “background-color”, “display”, “color”, “font-family”, “font-size”, “font-weight”, “padding”, and “text-decoration” properties for the “HistoryButton” class to make the buttons look like buttons. I used the “background-color” property to make the buttons have an opaque black color. Then, I set the “display” property to “inline-block” so that the buttons would appear next to each other. The “color”, “font-family”, and “font-size” properties change the font color, font family, and font size of the buttons. I then used “font-weight” to make the text bold and “padding” to make the button bigger. Lastly, I set the “text-decoration” property to none so that the default underline created by links in the browser would be removed.

CSS: 

.HistoryButton {
	background-color:rgba(0,0,0, 0.5);	
	display:inline-block;
	color:#ffffff;
	font-family:Arial;
	font-size:60px;
	font-weight:bold;
	padding:6px 20px;
	text-decoration:none;
}

To create the button background color change on hover, I created another "HistoryButton” class in CSS with a pseudo class of “:hover”. This lets me specify different properties for the buttons when I hover over them. In this pseudo class, I used the “background-color” and “color properties to set the background color to white and the text color to black on hover.

CSS: 

.HistoryButton:hover {
	background-color:white;
	color: black;
}

I also wanted to center the two buttons. To do this, I used my “buttons” class in my CSS. In this class, I set "text-align” to center to horizontally center the buttons. Then, I set the “position” property to “relative” so that I could move it from its original position. Lastly, I set the “top” property to 50% so the buttons would be vertically centered.

CSS: 

.buttons {
    text-align: center;
    position: relative;
    top: 50%;
}
