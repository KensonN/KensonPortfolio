Page Metadata

First, I needed to provide the metadata for my page. The only metadata I needed was the title of the page and the CSS file. The title tag is necessary in all HTML documents. It is the title that shows on the browser tab and comes up in search engine results.

HTML: 

<title>"Your Title Here"</title>

Since I had 3 different Hardware HTML files, I needed to set different titles for each of them. Using the <title> tag, I was able to do this. For the hardware redirect page, I set the title as “Hardware”. For the parts page, I set it as “Parts”. Lastly, for the History page, I set the title as “History”.

HTML: 

<title>"Hardware"</title>
<title>"Parts"</title>
<title>"History"</title>

To link the CSS file, I used the <link> tag. This tells the HTML file to use another CSS file for formatting. Since my CSS file was named “../../CSS/Styles.css, I used the link tag to link the HTML to that CSS file.

HTML: 

<link rel="stylesheet" href="../../CSS/Styles.css" type="text/css" />
