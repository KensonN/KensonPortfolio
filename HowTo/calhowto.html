Calculator

This is a page describing how the code behind the calculator on this website works. This page will be divided into 3 different parts: the HTML, the CSS, and the JavaScript of the page.

The HTML

The HTML of this page is very simple. We'll first be talking about the metadata of this page. The metadata is the information inside the head tags. The title tag of the page is the name of the tab. The link tag links the CSS file, named ../../CSS/Styles.css, to the HTML. 

<title>Calculator</title>
<link rel="stylesheet" href="../../../CSS/Styles.css" type="text/css" />

The rest of the HTML is the navigation bar and the formatting of the calculator. Since we have talked about the navigation bar in another page, we will instead talk about the calculator itself. The calculator is made up of 2 input boxes, one dropdown menu with the choice to choose and operation, and a "calculate" button to execute the problems.

First, we will talk about the input boxes on the page. Both are very simple. They are <input> tags, with seperate ID's for each of them. We will use these ID's later to make our calculator work. 

<input id="number1">
<input id="number2">

Now, we will talk about the dropdown menu. To create our dropdown menu, we use the <select> tag. Then, to create our options in the dropdown menu, we use the <option> tag. 

<select size="1" id="operation">
    <option>
        -Select-
    </option>
    <option>
        +
    </option>
    <option>
        -
    </option>
    <option>
        *
    </option>
    <option>
        /
    </option>
    <option>
        ^
    </option>
    <option>
        nth root
    </option>
</select>

The JavaScript

We will now talk about the JavaScript of the page. Since JavaScript and HTML work closely, we have already assigned ID's to HTML elements that will be needed for the calculator to work.

We are going to first talk about the variables we defined at the very beginning of the JavaScript file. Each of the variables represents an element of the HTML file, such as the input boxes and dropdown menu. We can link the variables to the HTML elements using the "document.getElementById" function in JavaScript.
