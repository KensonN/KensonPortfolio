Calculator

This is a page describing how the code behind the calculator on this website works. This page will be divided into 3 different parts: the HTML, the CSS, and the JavaScript of the page.

The HTML

The HTML of this page is very simple. We'll first be talking about the metadata of this page. The metadata is the information inside the head tags. The title tag of the page is the name of the tab. The link tag links the CSS file, named ../../CSS/Styles.css, to the HTML. 

<title>Calculator</title>
<link rel="stylesheet" href="../../../CSS/Styles.css" type="text/css" />

The rest of the HTML is the navigation bar and the formatting of the calculator. Since we have talked about the navigation bar in another page, we will instead talk about the calculator itself. The calculator is made up of 2 input boxes, one dropdown menu with the choice to choose and operation, and a "calculate" button to execute the problems.

First, we will talk about the input boxes on the page. Both are very simple. They are <input> tags, with seperate ID's for each of them. We will use these ID's later to make our calculator work. 

<input id="number1">
<input id="number2">

Now, we will talk about the dropdown menu. To create our dropdown menu, we use the <select> tag. Then, to create our options in the dropdown menu, we use the <option> tag. 

<select size="1" id="operation">
    <option>
        -Select-
    </option>
    <option>
        +
    </option>
    <option>
        -
    </option>
    <option>
        *
    </option>
    <option>
        /
    </option>
    <option>
        ^
    </option>
    <option>
        nth root
    </option>
</select>

The JavaScript

We will now talk about the JavaScript of the page. Since JavaScript and HTML work closely, we have already assigned ID's to HTML elements that will be needed for the calculator to work.

We are going to first talk about the variables we defined at the very beginning of the JavaScript file. Each of the variables represents an element of the HTML file, such as the input boxes and dropdown menu. We can link the variables to the HTML elements using the "document.getElementById" function in JavaScript. 

var number1 = document.getElementById("number1");
var operation = document.getElementById("operation");
var number2 = document.getElementById("number2");
var calculate = document.getElementById("calculate");

Next, we will talk about the event listener. This is an event listener to look for a click of the button with the id "calculate". Because of this, we start off our event listener statement with the variable calculate. Since we are also looking for a click, we also need to write "click" in our event listener to let the Javascript know what we are looking for is a click. Lastly, we write function so that we can contain code on what to do once the button is pressed. 

calculate.addEventListener("click", function(){

Inside the function, I first define a variable called "selected". I set this variable equal to the value of the variable "operation", which is the dropdown menu that allows for a user to choose a function. 

calculate.addEventListener("click", function(){
    var selected = operation.value

After defining a variable to detect the dropdown menu item, I created if/else statements to see what operation to do. For example, one statement says "if + is selected, then add the 2 numbers". For each of the operations, I had to use parseInt so that instead of just adding the strings together, it would add the numbers together instead. For the nth root, since the nth root is the same as 1 over the exponent, I used the exponent function of JavaScript. 

calculate.addEventListener("click", function (){
    var selected = operation.value; 
    if selected==="+") {
    sum = parseInt(number1.value) + parseInt(number2.value)
    }
    if selected==="-") {
    sum = parseInt(number1.value) - parseInt(number2.value)
    }
    if selected==="*") {
    sum = parseInt(number1.value) * parseInt(number2.value)
    }
    if selected==="/") {
    sum = parseInt(number1.value) / parseInt(number2.value)
    }
    if selected==="^") {
    sum = Math.pow(number1.value, number2.value)
    }
    if selected==="nth root") {
    sum = Math.pow(number1.value, 1/number2.value)
    }
})

In addition to just the operations, I made a check in each of the if statements to output to replace Infinity and -Infinity with "Number too big. Try again"
