What this is

This page will be detailing how one of the most crucial parts of the music player, the play button, worked.

HTML

The HTML for the HTML play button is extremely self explanatory. To do it, I used a button tag, with the id "play", to be referenced later in the JavaScript. I set the button to use the image "play.png", since this makes the button look like a play button. I also set the image to the id "playImage", to set the width of the image. 

//Play Button
<button id="play">
    <img id="playImage" src="icons/play.png">
</button>

The CSS

The CSS for the button is also extremely simple. Since HTML buttons, by default, have a border and background color, I needed to remove this to CSS. The play button is inside a div with the class "playButtons", so I set all buttons inside this div to have no background and border. Also, to set the width 

//Sets background and border to 0
.playButtons button {
    border: none; 
    background: none;
}
//Sets image to 80px wide
#playImage {
    width: 80px;
}

The JavaScript

The JavaScript is where the play button gets its functionality.

First, I created a variable, called playButton, which would fetch the element with the id of "play" from our HTML. 

//Fetches play button from HTML
var playButton = document.getElementById("play");

Then, I created an event listener, which would detect that if playButton is clicked, then run a function called playAudio. 

//Event listener to detect when play button is pressed
playButton.addEventListener("click", playAudio);

Lastly, I created the function playAudio. This function contains a simple if else statement. The function checks that if audio.paused, which means the audio is paused, then audio.play(), which makes the audio play. Also, change the image to the pause icon. Otherwise, pause the audio with audio.pause() and change the icon to a play button. 

function playAudio() {
    //if audio is paused
    if (audio.paused) {
    //play audio        
            audio.play();
    //change icon to pause        
        playImage.setAttribute('src', "icons/pause.png")
        }
    else {
    //pause audio        
        audio.pause();
    //change icon to play        
        playImage.setAttribute('src', "icons/play.png")
        }
}
