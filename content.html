Hardware and Parts of a Computer Page

This was the actual content of the hardware pages. I'm explaining both at the same time, since both are structured in the same way.

Navigation Bar (navbar)

The underlying HTML code of the navbar is simply an unordered list with anchor tags as list items. I created a div to contain the list, with the class “navbar”. Inside this div, I created an unordered list using the <ul> tag. I then used the <li> tag to create list items that contained anchor tags linking to other pages. With all of this done, the navbar just looks like a list of links.

HTML 

<ul class="navbar">
<li>
    <a href=index.html>
        Home
    </a>
</li>
<li>
    <a href="about.html">
        About
    </a>
</li>
<li>
    <a href="hardware.html">
        Hardware
    </a>
</li>
<li>
    <a href="https://www.youtube.com/watch?v=2HQaBWziYvY">
        Calculator
    </a>
</li>
<li>
    <a href="https://www.youtube.com/watch?v=ZbM6WbUw7Bs">
        Todo List
    </a>
</li>
<li>
    <a href="https://www.youtube.com/watch?v=SQoA_wjmE9w">
        Vindinium
    </a>
</li>
<li>
    <a href="https://www.youtube.com/watch?v=aoXDe8HxHBA">
        Big Project
    </a>
</li>
<li>
    <a href="howto.html">
        How-To
    </a>
</li>
</ul>

Because of this, we need CSS. In our CSS, we can define properties for our “navbar” class. In the CSS, I used the “background-color” property to set the background color to an opaque black. I also used the “border-radius” property to curze the corners of the nav bar. Lastly, I set the “font-size” property to 0 to reset the default browser formatting.

CSS 

.navbar {
    font-size: 0px;
    background-color: rgba(0, 0, 0, 0.5);
    border-radius: 10px; 
}

Also, to finally make the links appear in a line, I first had to create a class that included everything in the <li> tags. This is shown in the code with the greater than symbol. I used the “display” tag to make all the items inline and used the “width” tag to space the buttons out evenly.

CSS 

.navbar > li {
    width: 12.5%;
    display: inline-block;
}

For the text in the navbar, I created a selector that selected all of the anchor tags in the .navbar class. Then, I used some font properties to change the look of the links. One of the new tags used here is “text-decoration”. I set this to “none” so that the default link formatting would be removed. I also used some padding to space out the words to look more evenly distributed.

CSS 

.navbar a {
    text-decoration: none;
    color: white;
    font-family: 'roboto';
    font-weight: bold;
    font-size: 25px;
    padding: 0px 20px;
}

Lastly, since I wanted the navbar links to change background and text color when hovered over, I created a pseudo class “:hover”. I used the “color” property for the text and the “background-color” property for the background color. I also used the “border-radius” property so the white background behind the text during the hover seems rounded, matching the navbar.

CSS 

.navbar a:hover{
    background-color: white;
    border-radius: 8px;
    color: black;
}

Page Layout

On both hardware pages, there is a background image. I did this by adding the class “hardware” to the body tags of both pages. This class includes the “background-image” property, which sets the background image for the whole page.

HTML 

<body class="hardware">
</body>

CSS 

.hardware {
    background-image: url("VectorPhone.jpg");
}

Also, all of the content of the page (excluding the navbar) has a black background color. I did this by putting all of the content in a div with the class “text”. In this class, I used the property “background-color” to create a black opaque color. Then, I used the “padding” and “margin-top” properties to position the content correctly. Lastly, I used the “border-radius” to round the corners.

HTML 

<body class="hardware">
        <div class="text">
        </div>
</body>

CSS 

.text {
    background-color: rgba(0,0,0, 0.8);
    margin-top: -2%;
    padding: 0px 20px 20px 20px;
    border-radius: 50px;
}

Main Page Header

To create the main page header, I used an <h1> tag. Then, in CSS, I specified a few properties using the h1 selector. I changed the font size using the “font-size” property, reset the bottom margin by setting the “margin-bottom” property to 0px, changed the font to Roboto using the “font-family” property, created a text underline using the “text-decoration” property, and used the “color” to make the header white.

HTML 

<body class="hardware">
        <div class="text">
                <h1>
                    History<    
                /h1>
        </div>
</body>

CSS 

h1 {
    font-size: 100px;
    margin-bottom: 0px;
    font-family: roboto;
    text-decoration: underline;
    color: white;
}

Smaller Headers

For the smaller headers, I used the <h2> tag. The CSS properties were almost the same as the main header. However, I set the “font-size” property to 60, set the “margin-bottom” to 20px, and removed the “text-decoration” underline. Instead, I created a underline using the “border-bottom“ property.

HTML 

<body class="hardware">
    <div class="text">
        <h1>
            History
        </h1>
        <h2>
            Vacuum Tubes
        </h2>
    </div>
</body>

CSS 

h2{
    font-size: 60px;
    margin-bottom: 20px;
    font-family: roboto;
    border-bottom: 5px solid white;
    color: white;
}

Paragraphs

All of the paragraphs in the HTML are in paragraph tags. In these paragraph tags is the text and images of each section. In this section, we’ll talk about the text. In CSS, I created a selector that selects all of the paragraphs. In this selector, I used the “color” property to make the text white, set the font to Roboto using the “font-family” property, and set the font size to 30px using the “font-size” property.

HTML 

<body class="hardware">
    <div class="text">
        <h1>
            History
        </h1>
        <h2>
            Vacuum Tubes
        </h2>
        <p>
            stuff
        </p>
        </div>
</body>

CSS 

p{
    display: inline-block;
    font-family: roboto;
    color: white;
}

Images

For all of the images in both pages, I used a <img> tag that had the url of each image. In addition to the url, the tag also included a “width” property to size the image and a “align” property set to “right” to make the images align to the right. All of these image tags were placed before the text inside the paragraph tags. I did not use any CSS for the images.

HTML 

<body class="hardware">
    <div class="text">
        <h1>
            History
        </h1>
        <h2>
            Vacuum Tubes
        </h2>
        <img src="http://cdn.mos.cms.futurecdn.net/b5XerwXJrsM6HFEDS9LAP4.png" width=400px align="right"/>
        </div>
</body>
